## Tools used

1. **Adobe After Effect + Lottie** for the startup animation

2. **Slider** for slides

3. **AOS** for animations

4. **Font Awesome** for Icons

## Installation steps

1. Install all dependencies

``` sh 
npm install
```
2. Run app locally

``` sh 
npm run dev
```

3. Build for deployment

``` sh 
npm run build
```
